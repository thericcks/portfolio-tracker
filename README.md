# Portfolio Tracker

A full-stack portfolio tracking application built using React and Flask.

## Features

- Add investments
- Sector allocation visualization
- Investment type allocation
- Interactive charts
- Excel-based storage

## Tech Stack

Frontend:
- React
- Axios
- Recharts

Backend:
- Flask
- Pandas
- OpenPyXL

## Run Locally

### Backend

```bash
cd backend
python app.py
```

### Frontend

```bash
cd frontend
npm install
npm start
```
